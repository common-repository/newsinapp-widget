Newsinapp Widget finds latest news about any topics you choose and shows them directly on your wordpress website.

== Description ==

Newsinapp Widget adds a news feed on your web site. This news feed is automatically fed by latest news on topic you chosen.

For example if you have a cinema blog, newsinapp widget is perfect to show news about Tim Burton and Johnny Depp : latest movies, trailers, ...You can show news about any kind of topic, from Pedro Almodovar to Quantum field theory

Newsinapp Widget offers relevant news from Twitter, Youtube, Bing, Flickr and more.

== Installation ==

In order to use Newsinapp widget, you have to choose topics you want from newsinapp.io website. It's easy to setup, here is the detailed steps :

1. Register to newsinapp.io (it's quick and free) : [http://newsinapp.io/register/register/](http://newsinapp.io/register/register/)
2. Choose topics you want to show from [http://newsinapp.io/setup](http://newsinapp.io/setup)
2. Get your "Public Key" and "Private Key" from [http://newsinapp.io/setup/#settings](http://newsinapp.io/setup/#settings)
4. Install Newsinapp Widget from the WordPress admin : Visit the Plugins - > Add Newpage and search for 'Newsinapp'. Click to install.
5. Activate the plugin through the 'Plugins' menu in WordPress.
6. Create the widget from the Widget Page.
7. Write the Public and Private Key in the widget settings and save.
8. Enjoy!
